El header con class container-fluid para que ocupe el 10%% de la pantalla

Article

Todas las secciones y los divs se encuentran dentro de una section de class="container", el container es para crear todo un cuerpo

Este article, está dentro de un section. Al section se le puso clase main row. El main indica que es la sección principal, y row indica que se tomará como una fila de bootstrap

Aside

Esta región aside, y la anterior región Article forman las dos columnas de la fila declarada en section

Cada fila de bootstrap tiene 12 columnas.

Cada sección dentro de una fila podrá ocupar de 1 a 12 columnas

Pero el número de columnas utilizadas en total siempre deberá dar 12

Por ejemplo: este aside usa 3 columnas y el Article que le precede utiliza 9

Div A

Los siguientes div (A,B,C,D) pertenecen a otra fila

Cada uno deberá ocupar 3 columnas cuando el tamaño del monitor sea grande o extra-grande

Div B

Si la pantalla es muy pequeña o mediana, cada div utilizará el máximo número de columnas, 12

Decidir cuántas columnas ocupará una sección según el tipo de pantalla se logra con las siguientes clases:

    • col-xs-#: xs significa extra small, el # debe ser reemplazado por el número de columnas a ocupar
      col-md-#: md significa medium
      col-lg-#: ld significa large
      col-xl-#: xl significa extra large

    • Div C

    Antes de este div, existe un div class= clearfix visible-sm-block que permite que no se deforme el formato por ejemplo como aquí donde el div B es más grande que el div C
